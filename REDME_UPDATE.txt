📄 מדריך לעדכון האפליקציה (יצירת APK חדש)
בכל פעם שאת משנה משהו בקוד (מוסיפה פיצ'ר, משנה עיצוב, מתקנת באג), בצעי את השלבים הבאים לפי הסדר כדי ליצור קובץ התקנה מעודכן לטלפון:

שלב 1: עדכון הקוד

בצעי את השינויים הרצויים בקבצי ה-React ב-VS Code.

שמרי את כל הקבצים.

שלב 2: בניית האתר מחדש פתחי את הטרמינל ב-VS Code והריצי את הפקודה:

Bash

npm run build
(זה יוצר מחדש את תיקיית ה-dist עם השינויים שלך)

שלב 3: סנכרון לאנדרואיד הריצי את הפקודה הבאה כדי להעתיק את השינויים לתוך פרויקט האנדרואיד:

Bash

npx cap sync
שלב 4: יצירת קובץ ההתקנה (APK)

פתחי את Android Studio (אפשר דרך התפריט במחשב או עם npx cap open android).

חכי שהפסים למטה יסיימו לרוץ (Gradle Sync).

בתפריט העליון, לחצי על: Build -> Build Bundle(s) / APK(s) -> Build APK(s)

חכי שהבנייה תסתיים.

שלב 5: התקנה

בסיום, לחצי על הקישור הכחול locate שמופיע בבועה למטה. (או כנסי ידנית לתיקייה: android\app\build\outputs\apk\debug)

שלחי את הקובץ app-debug.apk לטלפון והתקיני אותו.
